# 天气机器人设计文档

**日期**: 2026-05-29
**项目**: 天气机器人 (Weather Robot)

---

## 1. 项目概述

开发一个每日天气播报聊天机器人，通过企业微信每天早上6点向用户发送天气消息，包含温度、湿度、风力等参数，并给出穿着建议和人性化吐槽。

---

## 2. 技术方案

**方案**: 企业微信自建应用 + 和风天气API

### 系统架构

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   和风天气API    │────▶│   天气机器人     │────▶│   企业微信      │
│   (免费API)     │     │   (定时任务)     │     │   (消息推送)    │
└─────────────────┘     └──────────────────┘     └─────────────────┘
                              │
                              ▼
                        ┌──────────────────┐
                        │   定时器 (APScheduler) │
                        │   每天早上6点触发    │
                        └──────────────────┘
```

### 技术栈
- Python 3.8+
- requests（HTTP请求）
- APScheduler（定时任务）
- 和风天气API（免费）

---

## 3. 核心组件

### 3.1 天气服务 (weather_service.py)
- 调用和风天气API获取实时天气数据
- 解析温度、湿度、风力、降雨概率等

### 3.2 消息生成器 (message_generator.py)
- 根据天气数据生成人性化消息
- 包含：天气描述、穿着建议、带伞提醒、吐槽
- 随机选择早上问候语

### 3.3 企业微信推送 (wechat_client.py)
- 调用企业微信应用消息API
- 推送到指定用户

### 3.4 定时调度器 (scheduler.py)
- 使用APScheduler设定每天6:00执行
- 处理定时触发逻辑

### 3.5 配置文件 (config.py)
- 集中管理所有配置（API密钥、企业微信参数等）

## 4. 消息格式

### 随机开头语（每日随机选一条）
1. 早上好，今天是元气满满的一天！来看看天气吧～
2. 早啊！睡得好吗？来瞅瞅今天的天气～
3. 新的一天，新的天气！让我来给你播报一下～
4. 起床啦！让我看看外面是什么天气～
5. 早起的鸟儿有虫吃，早起的你有天气看！
6. 嘿，醒醒！天气播报员准时上线～
7. 早上好呀！天气预报已新鲜出炉，请查收～
8. 新的一天开始了～让我来告诉你今天适不适合出门浪！

### 消息示例

```
☀️ 上海今日天气（5月29日 周五）

早上好，今天是元气满满的一天！来看看天气吧～

气温：26°C（今日高温28°C，低温19°C）
湿度：65%
风力：东南风3-4级

👔 穿着建议：今天阳光明媚，适合穿轻薄短袖或长裙，出门记得防晒哦～

🌧️ 降雨概率：20%，大概率不会下雨，但包里可以备把伞以防万一～

💬 这天气热得我都想原地蒸发了，但至少比上周的梅雨季强吧？且行且珍惜！
```

## 5. 配置需求

- 和风天气API Key（免费注册：https://dev.heweather.com/）
- 企业微信企业ID
- 企业微信应用AgentID
- 企业微信应用Secret
- 接收消息的成员账号（用户的企业微信账号）
