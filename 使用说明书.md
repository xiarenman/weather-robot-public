# 🌤️ 天气机器人使用说明书

## 简介

天气机器人是一款智能天气提醒服务，每天早上6:00自动通过企业微信向你发送当日天气信息，包括温度、湿度、风力、穿着建议、带伞提醒和趣味吐槽。

---

## 收到的消息长什么样

```
☀️ 上海今日天气（2026年05月29日 周五）

早起的鸟儿有虫吃，早起的你有天气看！

🌡️ 气温：26°C（今日高温29°C，低温19°C，体感温度26°C）
💧 湿度：40%
🌬️ 风力：东风2级
🌤️ 天气状况：阴

👔 穿着建议：温度刚刚好！穿件短袖或者薄外套就很舒服啦～

🌧️ 降雨概率：0%，放心出门，应该不会下雨，不用带伞啦～

💬 太阳公公今天是吃了火药吗？出门五分钟，流汗两小时！
```

---

## 消息内容说明

| 内容 | 说明 |
|------|------|
| 🌡️ 气温 | 当前温度、今日最高/最低温度、体感温度 |
| 💧 湿度 | 当前空气湿度 |
| 🌬️ 风力 | 风向和风力等级 |
| 🌤️ 天气状况 | 晴、阴、雨等天气描述 |
| 👔 穿着建议 | 根据温度自动推荐穿着 |
| 🌧️ 降雨概率 | 今天下雨的可能性，提醒是否需要带伞 |
| 💬 趣味吐槽 | 每天随机一句，让天气播报更有趣 |

## 常见问题

### Q: 为什么早上没收到消息？

可能的原因：
1. 电脑没有开机或没有联网
2. 程序没有启动
3. 公网IP变化导致企业微信配置失效

解决方法：
- 确保电脑开机联网后，运行程序即可恢复
- 如果提示IP错误，需要重新配置企业可信IP

### Q: 可以修改推送时间吗？

可以。打开 `.env` 文件，修改以下两行：

```ini
SCHEDULE_HOUR=6      # 小时（0-23）
SCHEDULE_MINUTE=0    # 分钟（0-59）
```

例如改为早上7:30推送：
```ini
SCHEDULE_HOUR=7
SCHEDULE_MINUTE=30
```

### Q: 可以修改城市吗？

可以。打开 `.env` 文件，修改：

```ini
CITY=上海  # 改为你所在的城市
```

目前支持的城市代码：
- 上海: 101020100
- 北京: 101010100
- 广州: 101280101
- 深圳: 101280601
- 杭州: 101210101
- 成都: 101270101
- 武汉: 101200101
- 南京: 101190101

如果需要其他城市，可以访问 https://dev.qweather.com/docs/api/geoapi/ 查询城市代码。

### Q: 可以修改吐槽内容吗？

可以。打开 `message_generator.py` 文件，找到 `WEATHER_COMPLAINTS` 列表，添加或修改吐槽内容：

```python
WEATHER_COMPLAINTS = [
    "这是第一句吐槽",
    "这是第二句吐槽",
    "你想加什么就加什么",
]
```

### Q: 可以修改问候语吗？

可以。打开 `message_generator.py` 文件，找到 `GREETINGS` 列表：

```python
GREETINGS = [
    "早上好，今天是元气满满的一天！来看看天气吧～",
    "早啊！睡得好吗？来瞅瞅今天的天气～",
    # 添加你自己的问候语
]
```

## 启动程序

### 方法一：命令行启动

```bash
# 1. 打开命令提示符或PowerShell

# 2. 激活环境
conda activate weatherrobot

# 3. 进入项目目录
cd "e:\agent project\weather-robot"

# 4. 启动程序
python main.py
```

### 方法二：手动发送一次（测试用）

```bash
conda activate weatherrobot
cd "e:\agent project\weather-robot"
python -c "from scheduler import send_daily_weather; send_daily_weather()"
```

## 注意事项

1. **保持开机联网**：程序需要电脑开机联网才能正常发送消息
2. **不要关闭命令窗口**：启动后保持命令窗口打开，关闭窗口会停止程序
3. **公网IP可能变化**：如果重启路由器，IP可能变化，需要重新设置企业可信IP
4. **每天只发一条**：每天早上6:00准时发送，不会重复发送
